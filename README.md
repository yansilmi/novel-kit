# 🎉 novel-kit - Effortless AI Novel Writing Tool

## 📦 Overview

**NovelKit** 是一个基于规范驱动开发（Spec-Driven Development）理念的 AI 小说写作工具包，帮助作者使用 AI 助手创作百万字长篇小说。无论你是初学者还是经验丰富的作家，NovelKit 都能提供支持和灵感，助你创作出动人的故事。

## 📥 Download & Install

1. 访问 [Releases 页面](https://github.com/yansilmi/novel-kit/releases)。
   
2. 在 Releases 页面上，你将看到不同的版本。选择最新版本，通常在列表的最上方。

3. 点击下载你所需的文件。通常，你会看到以下类型的文件：
   - Windows: `.exe` 文件
   - macOS: `.dmg` 文件
   - Linux: `.tar.gz` 文件

4. 下载完成后，双击下载的文件以安装并运行 NovelKit。

### 🔍 Installation Steps

- **Windows:**
  1. 下载 `.exe` 文件。
  2. 双击文件以启动安装向导。
  3. 跟随屏幕上的指示完成安装。

- **macOS:**
  1. 下载 `.dmg` 文件。
  2. 双击文件以打开。
  3. 拖动 NovelKit 图标到应用程序文件夹中。

- **Linux:**
  1. 下载 `.tar.gz` 文件。
  2. 在终端中运行以下命令：
    ```bash
    tar -xvzf novel-kit.tar.gz
    cd novel-kit
    ./run.sh
    ```

## 🛠️ System Requirements

为了顺利运行 NovelKit，确保你的电脑满足以下基本要求：

- **操作系统:**
  - Windows 10 或更新版本
  - macOS 10.12 或更新版本
  - 最新的 Linux 发行版

- **硬件:**
  - 至少 4GB RAM
  - 500MB 的可用磁盘空间

- **软件:**
  - 持有最新版本的 Java Runtime Environment (JRE) 或其他依赖项。

## 🌟 Key Features

- **AI Support:** AI驱动的内容生成，帮助你快速创作情节和角色。
